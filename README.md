# 📊 Education in India Data Analysis Workbook

A comprehensive Python data analysis workbook based on the **Education in India (DISE 2015–16)** dataset. This project contains solutions to **50 practical exercises** covering data exploration, preprocessing, statistical analysis, visualization, and reporting using real-world educational data.

---

## 📌 Project Overview

This notebook demonstrates the complete data analysis workflow using Python. The exercises gradually progress from basic dataset exploration to advanced analytical techniques and visualizations.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

- **Dataset:** Education in India (DISE 2015–16)
- District-wise educational statistics across India
- Includes population, literacy, schools, teachers, enrollment, infrastructure, and other educational indicators.
